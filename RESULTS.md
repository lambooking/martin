## 1. Linux 下查看 TIF 文件

### 方式 A：Python 快速预览（推荐，已内置）

```bash
# 查看单期（自动选第一期有数据的期次）
python martin/quick_preview.py

# 指定期次：2 行 10 图（B1/B2/B4/B5/B6 栅格 + B7/C1/C3/D/C4 矢量）
python martin/quick_preview.py --period 2019_Q1

# 批量生成全部 24 期
python martin/quick_preview.py --all

# 仅矢量总览（断面网格 + 6年岸线演变 + 热点）
python martin/quick_preview.py --vector
```

输出位置：`output/figures/preview_{period}.png` 和 `output/figures/preview_vectors.png`

### 方式 B：QGIS（GUI，推荐用于精细检查）

```bash
# 安装（如未安装）
sudo apt install qgis

# 打开单个 TIF
qgis output/sea_mask/2019_Q1_sea.tif
```

打开后：图层右键 → **Properties → Symbology** → 将 Min/Max 设为 `0 / 255`，可清晰看到：
- `255`（白）= 水体 / 海水
- `0`（黑）= 陆地
- `128`（灰）= 无效像元（nodata）

可同时将 `.gpkg` 矢量文件（waterline、transects、hotspots 等）拖入叠加显示。

### 方式 C：GDAL 命令行

```bash
# 查看元数据（波段数、分辨率、CRS、像素值范围）
gdalinfo output/sea_mask/2019_Q1_sea.tif

# 快速转 PNG（自动拉伸到 0-255）
gdal_translate -of PNG output/sea_mask/2019_Q1_sea.tif /tmp/preview.png
eog /tmp/preview.png   # GNOME 图像查看器

# 批量转 PNG
for f in output/sea_mask/*.tif; do
    gdal_translate -of PNG "$f" /tmp/"$(basename $f .tif)".png
done
```

### B5：形态学清洁

- 开运算（去碎斑）+ 闭运算（填洞）+ 面积过滤（< 最小面积阈值的斑块移除）
- 24 期均正常完成

### B6：外海连通约束

- 通过种子点（SEA_SEED_COORDS）的 flood-fill 只保留与外海相连的水体
- 平均移除约 **10.6%** 的内陆水体像元
- ⚠️ **2022_Q1 异常**：移除率高达 **54.1%**，疑为种子点落于海陆边界外侧，待 QGIS 核查
